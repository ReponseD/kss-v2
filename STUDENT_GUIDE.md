# Student Guide: Using Netlify CMS

## Quick Start

Welcome to the Kagarama Secondary School Content Management System! This guide will help you create and manage content for the school website.

**CMS URL:** https://www.kagaramasec.org/admin

---

## Getting Started

### First Time Login

1. Visit https://www.kagaramasec.org/admin
2. Click "Log in" or "Sign up"
3. Enter your email and password
4. Check your email for verification (if required)
5. You're ready to start creating content!

---

## Creating Your First Post

### Step-by-Step: Writing a News Article

1. **Click "Posts / Articles"** in the left sidebar
2. **Click "New Post / Article"** button (top right)
3. **Fill in the form:**
   - **Title**: "School Wins Science Competition"
   - **Author**: Your name (e.g., "John Doe, Media Club")
   - **Publish Date**: Today's date and time
   - **Featured Image**: Click to upload a photo
   - **Body**: Write your article here
   - **Category**: Select "News", "Blog", or "Announcement"
   - **Featured**: Check this box to show on homepage
4. **Preview**: Click "Preview" to see how it looks
5. **Publish**: Click "Publish" when ready

### Writing Tips

- **Use clear headings**: Start with `#` for main title, `##` for sections
- **Add images**: Break up text with relevant photos
- **Keep it concise**: Aim for 300-500 words
- **Check spelling**: Proofread before publishing
- **Use lists**: Make information easy to scan

## Adding Videos

### How to Add a YouTube Video

1. **Click "Media / Videos"** in the sidebar
2. **Click "New Media / Video"**
3. **Fill in:**
   - **Title**: "School Sports Day Highlights"
   - **Video URL**: Copy the YouTube link (e.g., `https://www.youtube.com/watch?v=...`)
   - **Video Type**: Select "YouTube"
   - **Description**: Write what the video is about
   - **Thumbnail**: Optional - upload a custom thumbnail
4. **Click "Publish"**

### Getting YouTube Video URL

1. Go to YouTube and find your video
2. Click "Share" below the video
3. Copy the link
4. Paste it into the "Video URL" field

---

## Creating Events/Announcements

### Step-by-Step: Creating an Event

1. **Click "Announcements / Events"**
2. **Click "New Announcement / Event"**
3. **Fill in:**
   - **Title**: "Parent-Teacher Meeting"
   - **Event Date**: When the event starts
   - **End Date**: When it ends (optional)
   - **Location**: "School Auditorium"
   - **Description**: Full details about the event
   - **Image**: Upload event flyer or photo
   - **Event Type**: Select appropriate type
4. **Click "Publish"**

---

## Editing Existing Content

1. Click on the collection (Posts, Media, or Events)
2. Find the item you want to edit
3. Click on it to open
4. Make your changes
5. Click "Save" or "Publish"

## Markdown Basics

Markdown is a simple way to format text. Here are the basics:

### Headings
```
# Main Title (largest)
## Section Title
### Subsection
```

### Bold and Italic
```
**bold text**
*italic text*
```

### Lists
```
- Item 1
- Item 2
- Item 3

1. First item
2. Second item
3. Third item
```

### Links
```
[Link text](https://example.com)
```

### Images
```
![Image description](/assets/uploads/image.jpg)
```

## Getting Help

### If You're Stuck

1. **Ask a teacher**: They can help with content
2. **Check this guide**: Review the steps
3. **Try preview mode**: See how content looks
4. **Save as draft**: Work on it later

### Common Questions

**Q: Can I edit content after publishing?**  
A: Yes! Just open the item and make changes.

**Q: What if I make a mistake?**  
A: You can always edit or delete content.

**Q: How do I add photos?**  
A: Click the image field and upload from your computer.

**Q: Can I save without publishing?**  
A: Yes, use "Save" for drafts, "Publish" to go live.
